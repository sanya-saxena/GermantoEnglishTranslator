This project deals with German to English Translation. It uses Keras for the same. 
This project is made in python and recommends Jupyter Notebook to be installed in your systems.
The project requires following packages to be installed in python:
	1. Python 3 SciPy environment installed.

	2. Keras (2.0 or higher) installed with either the TensorFlow or Theano backend.

	3.  NumPy 

	4. Matplotlib installed.
